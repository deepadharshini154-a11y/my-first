About the app

This application is a lightweight movie rating tool built with Flask. It provides a clean landing page, a separate login form, and a ratings page that lets you add and review movie scores.

Technology used

Flask

Flask is the Python web framework used to build routing, form handling, and template rendering.

HTML & CSS

The landing page, login page, and ratings interface are built using standard HTML and lightweight CSS styling.

Python

Python handles server-side logic, form processing, and simple in-memory storage of movie ratings.
